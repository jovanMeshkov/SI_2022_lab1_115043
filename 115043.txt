git init
- креира нова датотека за потребите на git

git remote add origin "<remote-url>"
- го регистрира оддалечениот репозиториум со локалниот фолдер

git pull
- собира податоци за гранките и повлекува промени од оддалечниот репозиториум

git add .
- ги регистрира сите датотеки и фолдери од локалниот репозиториум за соодветно верзионирање со одалечениот репозиториум

git commit -m "<message>"
- ги поврзува локално тековните промени за синхронизација со одалечниот репозиториум

git checkout -b <branch-name>
- прави нова локална гранка

git commit -m "Change quantity of item #5"

git push origin

git push origin branch1

git push origin branch2

git add .

git commit -m "Modify item #4"

git add .

git commit -m "Modify item #4 and add item #6"
